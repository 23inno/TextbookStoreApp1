# Textbook Store App

## Project Overview
Textbook Store App is an Android mobile application developed for students in higher education institutions. The app allows students to browse, search, view, and list second-hand textbooks.

## Purpose
The purpose of the application is to make it easier for students to access affordable textbooks through a simple peer-to-peer textbook marketplace.

## Target Users
- University students
- Students selling used textbooks
- Students looking for affordable academic resources

## Main Features
- User registration
- Textbook listing catalogue
- Search and filtering functionality
- Book details screen
- Sell book form
- Contact seller enquiry screen
- Profile screen
- RecyclerView for textbook listings
- Data Binding implementation
- Input validation

## Technologies Used
- Android Studio
- Kotlin
- XML
- RecyclerView
- Data Binding
- Git and GitHub

## Project Structure
The project is organised using separate folders for activities, adapters, models, and layout resources.
